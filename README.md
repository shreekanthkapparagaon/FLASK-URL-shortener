Clone the repo and get cozy:
```bash
git clone https://github.com/shreekanthkapparagaon/FLASK-URL-shortener.git
cd FLASK-URL-shortener
```

Create and activate a virtual environment:
```bash
python -m venv venv
# macOS/Linux

```

```bash
python main.py
```
